# クイックスタートガイド 🚀

## AIに依頼するだけでOK！

このツールは**AIアシスタント（Claude等）に依頼するだけ**で使えます。
コマンドを覚える必要はありません。

---

## 使い方

### 1. PDFファイルを `input/` フォルダに入れる

処理したいPDFファイルを `input/` フォルダにドラッグ&ドロップしてください。

### 2. AIに依頼する

以下のように依頼するだけです：

**基本的な依頼:**
> 「inputフォルダにあるPDFの余白を除去して」

**自動処理:**
> 「inputフォルダのPDFを自動で処理して」

**AI確認付き（推奨）:**
> 「PDFの余白を除去したい。まずプレビューを見せて」

### 3. 結果を受け取る

処理されたPDFは `output/` フォルダに保存されます。

---

## AIへの依頼例

| やりたいこと | AIへの依頼 |
|-------------|-----------|
| PDFの余白を除去したい | 「PDFの余白を除去して」 |
| Kindle向けに最適化したい | 「PDFをKindle用に最適化して」 |
| スキャンPDFを処理したい | 「これはスキャンPDFなので、それに合わせて処理して」 |
| 複数のPDFを一括処理 | 「inputフォルダのPDFを全部処理して」 |
| クロップ位置を自分で確認したい | 「まずプレビュー画像を見せて、それからクロップ位置を一緒に決めたい」 |
| Kindle Oasis用に最適化 | 「Oasis用に最適化して」 |

---

## トラブルシューティング

### 「コマンドが見つからない」と言われた
→ AIに「まずツールをインストールして」と依頼してください

### 余白がうまく取れない
→ AIに「もう少し余白を削って」または「文字が切れないようにして」と依頼してください

### スキャンPDFがうまく処理されない
→ AIに「スキャンPDFモードで処理して」と依頼してください

## フォルダ構成

```
epub/
├── input/     ← ここにPDFを入れる
├── output/    ← 処理済みPDFが出力される
├── preview/   ← プレビュー画像が出力される
└── ...
```
